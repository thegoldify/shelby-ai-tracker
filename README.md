# 🚀 Shelby Ecosystem Health Tracker

Bu proje, **Shelby AI** ağının performansını izlemek ve topluluğa katkı sağlamak amacıyla oluşturulmuş bir ön sunum projesidir. Erken erişim talebi için geliştirilmiştir.

## 📊 Proje Hakkında
Shelby ekosistemine duyduğum ilgiyle, ağın stabilitesini ve veri akışını simüle eden bu takip aracını hazırladım. Amacım, Shelby teknolojisini test etmek ve node operatörü olarak ekosisteme değer katmaktır.

- **Dil:** Python / HTML
